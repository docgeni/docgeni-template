 ## 快速上手

切换到已有的项目中，执行如下命令：

```bash
$ npx @docgeni/cli init
# 或者 
$ docgeni init 
# 或者
$ ng add @docgeni/cli
```
初始化后，使用`npm run start:docs`启动文档站点，浏览器打开`http://127.0.0.1:4600` 即可访问。

点击 [快速上手](https://docgeni.org/guides/intro/getting-started) 了解更多。
